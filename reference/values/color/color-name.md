#Color Name

##说明

##示例

```html

<!DOCTYPE html>
<html lang="zh-cmn-Hans">
<head>
<style>
.foreground { color: highlight; }
.background { background-color: gray; }
</style>
</head>
<body>
<ul class="test">
	<li class="foreground">红色的文字</li>
	<li class="background">灰色的背景</li>
</ul>
</body>
</html>

```
